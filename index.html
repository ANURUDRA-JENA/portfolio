Projects

Customer Segmentation-Unsupervised ML

> Had to analyze transactional data from a UK-based online retail business to optimize marketing and sales strategies.
> This project aims to analyze transactional data to gain insights into customer behavior, product performance, and temporal trends, providing actionable recommendations for business optimization.

Movie Recommender System; The classic ML Project

> This is the ultimate classic ML project for all ML enthusiasts. But to use the movie recommender system to filter out movies which I would like, I have created my custom dataset of movies from year 2013-2023, apart from IMDB's 1984-2013 dataset.
> Have used my proficiency in Web-Scraping tools to scrape the latest and updated movies along with their reviews and other information.
> The good'ol LSTM algorithm will recommend top 5 movies based on the input. If the recommender really works, I'll deploy it with good frontend features.

SW Airlines Co.: Airline Analysis Dashboard

> This project aims to analyze flight trends in terms of traffic, delays and cancellations.
> A visually appealing, self-intuitive, dynamic dashboard has been made which consists of overview, flight traffic analysis and time series analysis.

American Board of Hospital Medicine: Health Care Analysis Dashboard

> Project aims to leverage Power BI for a deep dive into the provided healthcare datasets.
> This task encompasses meticulous data cleaning and sophisticated data modeling, utilizing DAX for advanced analytics.
> This dashboard would serve as a tool to uncover and visualize important trends, such as the interplay between patient demographics and treatment outcomes, cost implications of various medical procedures, and overall hospital performance metrics.

Superstore: Sales Analysis using SQL

> This is an analysis based on insights generated by using SQL on the famous Superstore dataset.
> This analysis sheds light on insights based on sales and profits, regional performance, product performance and customer segmentation.

Workforce Absenteeism: Cohort Analysis

> The analysis aims to uncover patterns in workforce callouts, identify frequent absentees, and suggest improvements for workforce management.
> Highlights include the predominance of unscheduled sick leave, disparities in job roles, and shift-wise absenteeism patterns.
> Leveraged proficiency along tools like Pandas and Numpy along with Matplotlib to wrangle the data and generate insights.

AirBnb: Hotel Booking Analysis Dashboard

> This Analysis seeks to leverage Tableau Desktop/Tableau Public/PowerBI's visual analytics capabilities to uncover and illustrate the shared attributes, disparities, and distinctive patterns inherent to Airbnb's presence in these cities, thus elevating the depth and insightfulness of the study.
> Deployed a highly intuitive, fast and efficient dashboard using Tableau Public.

eBay: Business Case study to increase revenue
